Download
========

PyLit
-----

There are no file releases yet. Get the sources from the `SVN repository`_:

* `pylit.py`_:         text <--> code converter module
* `simplestates.py`_:  simple state machine framework 

Pylit needs a functioning Python_ installation. If it is missing on your
machine, you can get it from http://www.python.org/download/

Tools
-----

* docutils_: conversion of the text source to HTML or PDF formats (not
  stricly required by PyLit)
            
* `Editor Support for reStructuredText`_

* `pylit functions`_ for the `JED editor`_

.. _`SVN repository`: http://developer.berlios.de/svn/?group_id=7974
.. _`pylit.py`: http://svn.berlios.de/svnroot/repos/pylit/trunk/src/pylit.py
.. _`simplestates.py`: http://svn.berlios.de/svnroot/repos/pylit/trunk/src/simplestates.py
.. _Python: http:www.python.org
.. _docutils: http://docutils.sourceforge.net/
.. _`Editor Support for reStructuredText`: 
     http://docutils.sourceforge.net/tools/editors/README.html
.. _`pylit functions`: http://jedmodes.sourceforge.net/mode/pylit/     
.. _`JED editor`: http://www.jedsoft.org/jed/
