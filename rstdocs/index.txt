Welcome at the PyLit home page. Here you will find information and

`Literate Programming`__
  Definition and existing frameworks
    
    The idea is that you do not document programs (after the fact), but
    write documents that *contain* the programs.

    -- John Max Skaller in a `Charming Python interview`_.

__ literate-programming/index.html
.. _Charming Python interview:
     http://www-128.ibm.com/developerworks/library/l-pyth7.html

`Pylit Features`__
  PyLit (Python-Literate) is a bidirectional converter between 
 
  * a *text source* (`reStructured Text`_ with embedded code blocks) and
  * a *code source* (source code with comment blocks).
   
  Documentation or code can be added or edited in a "native" format using
  ones favourite editor and editing mode.

`Download, Installation, Usage`__
  PyLit requires functioning Python_ installation and the `pylit.py`_ and
  `simplestates.py`_ modules.
  
  Installation_ and usage_ are straigtforward.
  
  Not required by PyLit but recommended for conversion of the text source
  to HTML or PDF formats are the Python docutils_
