============================================================
PyLit: Literate programming with reStructuredText
============================================================


    The idea is that you do not document programs (after the fact), but
    write documents that *contain* the programs.

    -- John Max Skaller in a `Charming Python interview`_.

.. _Charming Python interview:
     http://www-128.ibm.com/developerworks/library/l-pyth7.html


Welcome at the PyLit home page. Here you will find information and
material for the following topics:

`Pylit Features`__ 
  `PyLit` (Python-Literate) is a `pure Python module`__ providing a
  bidirectional text <--> code converter. 
  
  A literate program source can be either in "text" or "code" format.
    
  * The "code" source can be debugged, compiled or executed with standard
    tools.
  
  * The "text" source can be converted to HTML or LaTeX documents with the
    Python Docutils_.

`Download, Installation, Usage`__
  PyLit requires functioning Python_ installation and the `pylit.py`_  module.
  Installation_ and usage_ are straigtforward.
  
  Docutils_ are not strictly required by PyLit itself but for conversion
  of the text source to HTML, LaTeX, or PDF formats.
